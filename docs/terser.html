Why ?

Rollup Plugin Terser is a good old companion of Rollup but, curiously, as I write these lines (in October 2022), it still does not support rollup 3.
Here is a fork I made to correct this.

Installation

npm

npm i -D rollup-plugin-terser-amstramgram

Manually

Download the plugin in a amstramgramRollupPluginTerser folder located at the root of your project.

Dependencies

If you have chosen the manual way, install also @babel/code-frame, jest-worker, and serialize-javascript. 

npm i -D @babel/code-frame jest-worker serialize-javascript
